## Descrição

  Este projeto utiliza as seguintes tecnologias:
  
  Back-End
  * Java 11
  * Spring Framework
  * H2 Database
  * MySQL
  * Maven
  
  Front-End
  * Typescript
  * React JS
  * Bootstrap
  

## Para executar 
  
 Para executar o basta o comando abaixo:
  ```sh
  ./mvnw clean test
  ``` 

  ```sh
  yarn start
  ```
  
  Base de dados para ser usada: [datasets.json](./src/test/java/br/com/supera/game/store/ProductDaoExampleTest.java)
